# epictoolkit CLI

A CLI created by Epic Quest TI for internal use by the Epic Quest TI team

### Installation

#### npm i -g epictoolkit

### Available commands

To run a command start with <code>etk</code> followed by one of the following commands and options

```shell
$ npm login
$ npm whoami
$ npm test

$ npm run build

$ npm publish
```
